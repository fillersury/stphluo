## Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/fillersury/stphluo.git
   cd stphluo
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
